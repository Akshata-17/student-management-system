# Student Management System

A Python-based application to manage student records using MySQL.

## Features
- Add student details
- View student records
- Update student information
- Delete student records
- Database connectivity using MySQL

## Technologies Used
- Python
- MySQL

## Purpose
This project was built to understand CRUD operations, database connectivity, and backend logic.

## How to Run
1. Install Python
2. Install MySQL
3. Configure database connection in the code
4. Run the Python file
